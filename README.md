## Getting Started

install dependencies:

```bash
npm run install
# or
yarn install
```

build project. This command generated static pages
You can start web server for folder `.next/server/pages`

```bash
npm run build
# or
yarn build
```

start dev project

```bash
npm run dev
# or
yarn dev
```

start ssr mode

```bash
npm run start
# or
yarn start
```
